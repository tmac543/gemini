# 手势控制使用指南

## 功能概述
Magic Christmas Tree 现在支持通过摄像头手势控制 3D 场景！

## 如何启用手势控制

1. **上传照片**
   - 在首页点击上传区域或拖拽照片
   - 上传 1-20 张照片
   - 点击"Decorate Tree"按钮

2. **启用手势识别**
   - 进入 3D 场景后，点击右上角的"启用手势"按钮
   - 浏览器会请求摄像头权限，点击"允许"
   - 等待初始化完成（右下角状态指示器会显示"已检测"）

## 支持的手势

### ✊ 握拳旋转
- **动作**: 握紧拳头，左右移动手臂
- **效果**: 旋转圣诞树
- **技巧**: 保持拳头握紧，手臂移动幅度不要太大

### 🤲 双手缩放
- **动作**: 双手同时伸入画面，张开或合拢
- **效果**: 放大或缩小圣诞树
- **技巧**: 
  - 双手距离越远 = 放大
  - 双手距离越近 = 缩小

### ✋ 张开手掌触发特效
- **动作**: 张开手掌，五指分开
- **效果**: 圣诞树会轻微弹跳
- **技巧**: 保持手掌完全张开 1-2 秒

## 状态指示器

右下角的状态面板会显示：
- 🟢 绿色圆点 = 手部已检测
- 🔴 灰色圆点 = 未检测到手部
- 🟡 黄色闪烁 = 正在初始化

## 故障排除

### 摄像头无法访问
- 检查浏览器权限设置
- 确保没有其他应用占用摄像头
- 刷新页面重试

### 手势识别不准确
- 确保光线充足
- 手部完全在摄像头视野内
- 背景尽量简洁
- 与摄像头保持 30-50cm 距离

### 想切换回鼠标控制
- 点击右上角的"手势已启用"按钮即可关闭
- 关闭后会自动恢复 OrbitControls 鼠标控制

## 技术实现

- **手势识别**: MediaPipe Hands
- **3D 引擎**: React Three Fiber
- **平滑插值**: Linear Interpolation (Lerp)
- **性能优化**: 60fps 实时渲染
