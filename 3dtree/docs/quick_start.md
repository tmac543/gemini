# 快速开始指南

## 🚀 5 分钟上手

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 3. 上传照片
1. 访问 http://localhost:5173
2. 点击上传区域或拖拽照片
3. 上传 1-20 张照片
4. 点击 **"Decorate Tree"** 按钮

### 4. 探索 3D 场景

#### 鼠标控制（默认）
- **左键拖拽** - 旋转视角
- **滚轮** - 缩放
- **右键拖拽** - 平移

#### 手势控制（可选）
1. 点击右上角 **"启用手势"** 按钮
2. 允许浏览器访问摄像头
3. 等待初始化完成（右下角显示"已检测"）

**支持的手势**:
- ✊ **握拳左右移动** → 旋转圣诞树
- 🤲 **双手张开/合拢** → 缩放场景
- ✋ **张开手掌** → 触发弹跳特效

## 📱 浏览器要求

### 推荐浏览器
- ✅ Chrome/Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+

### 手势控制要求
- ✅ 支持 WebRTC
- ✅ 摄像头权限
- ✅ 充足的光线
- ✅ 简洁的背景

## 🎯 最佳体验建议

### 照片选择
- 📸 使用清晰的照片
- 📸 人像照片效果最佳
- 📸 避免过暗或过亮的照片
- 📸 建议上传 10-20 张

### 手势识别
- 💡 确保光线充足
- 💡 手部完全在摄像头视野内
- 💡 背景尽量简洁
- 💡 与摄像头保持 30-50cm 距离
- 💡 动作不要太快

### 性能优化
- 🚀 关闭不必要的浏览器标签页
- 🚀 使用独立显卡（如有）
- 🚀 确保浏览器硬件加速已启用

## 🐛 常见问题

### 摄像头无法访问
**问题**: 点击"启用手势"后没有反应  
**解决**:
1. 检查浏览器权限设置
2. 确保没有其他应用占用摄像头
3. 刷新页面重试
4. 尝试使用 HTTPS（某些浏览器要求）

### 手势识别不准确
**问题**: 手势控制不灵敏  
**解决**:
1. 确保光线充足
2. 手部完全在摄像头视野内
3. 背景尽量简洁
4. 与摄像头保持合适距离
5. 动作幅度适中

### 帧率较低
**问题**: 画面卡顿  
**解决**:
1. 关闭其他占用资源的程序
2. 降低浏览器窗口大小
3. 关闭手势控制（使用鼠标）
4. 检查是否使用独立显卡

### 照片显示模糊
**问题**: 装饰球上的照片不清晰  
**解决**:
1. 上传更高分辨率的照片
2. 确保照片本身清晰
3. 尝试缩放到更近的距离查看

## 💡 提示

### 第一次使用
1. 先用鼠标控制熟悉场景
2. 再尝试手势控制
3. 在光线充足的环境下使用
